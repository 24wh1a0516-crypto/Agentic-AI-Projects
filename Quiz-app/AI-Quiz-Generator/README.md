# AI Quiz Generator

A Python-based starter project for generating quizzes from PowerPoint presentations using an LLM.

## Structure

- Upload a PPT or PPTX file
- Extract slide text
- Build prompts for quiz generation
- Generate multiple-choice questions
- Score quiz responses

## Setup

1. Create a virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Add your DeepSeek API key to `.env`
4. Run: `python main.py`
